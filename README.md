# 图片批量缩放工具

## 功能特性
- 递归处理指定目录及所有子目录的图片
- 支持格式：JPG/JPEG/PNG/BMP/GIF
- 图形化操作界面
- **两种缩放模式**：
    - **按比例缩放**：可调节缩放比例（默认0.5）
    - **固定分辨率**：可指定目标宽度和高度
- 实时处理进度提示
- 自动错误处理机制

## 环境要求
- Python 3.6+
- Pillow 库

## 快速开始
1. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
2. 运行图形界面：
   ```bash
   python gui.py
   ```
3. 在图形界面中选择图片所在目录。
4. 选择缩放模式（按比例或固定分辨率）并输入相应参数。
5. 点击“开始处理”按钮。

## 注意
- 处理会直接覆盖原文件，请提前备份重要图片。
